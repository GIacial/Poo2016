Installation : 

Dézipper le fichier
Créer un dossier Save à la base du dossier principal
Avoir java d'installé 


Utilisation : 

Lancer l'executable "executable.bat"
Afin de connaitre toutes les commandes taper "help" (c'est qui help ? )
Pour avoir un détail des commandes taper "help [le_nom_de_la_commande]" (sans les crochets)
